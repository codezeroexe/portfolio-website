Build a portfolio website for a full-stack developer named 'Hari'. Purpose: showcase coding projects and technical skills. Core Features: Hero with avatar, name, title, and CTA buttons (View My Work, Get In Touch) plus dark/light theme toggle; About section with bio and tech stack badges; Project grid with 3 project cards showing title, description, tags, GitHub repo and live demo links (no images); Contact section with email button and social icons (GitHub, LinkedIn, Instagram, X); Footer with copyright. Visual Vibes: Modern, clean. Colors: OKLCH-based (Tailwind v4 defaults), dark mode support with next-themes. Fonts: Inter (sans) + JetBrains Mono (mono) via next/font/google. Tech Stack: Next.js 16 (App Router), TypeScript, Tailwind v4, Shadcn UI (base-ui components), Remix Icons for social icons, Framer Motion for animations, next-themes for theme toggling. Components: Custom ProjectCard, SocialButtons, ThemeToggle; Shadcn UI components (Button, Card, Badge, Avatar, Separator). Deployed on Vercel. Repository: https://github.com/codezeroexe/portfolio-website (main branch).
